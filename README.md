# Taparia: Legendary Web Lab

![Total lines](https://img.shields.io/tokei/lines/github/vityaman-edu/semt-taparia-ant)

## Build & Run

Install requirements:

- `Java 17+`
- `Maven`
- `Docker`
- `make`
- `NodeJS npm`

Build:

```bash
sudo make wars
```

Build & Run:

```bash
sudo make infra
```
